KMF算法

programm
更新日期:   2024-09-04
文章字数:   1.6k
阅读次数:  

描述

给定一个 haystack 字符串和一个 needle 字符串，在 haystack 字符串中找出 needle 字符串出现的第一个位置 (从0开始)。如果不存在，则返回 -1。

注意

相较于leetcode 76 最小子串，这道题需要在主串（haystack 字符串）中寻找连续完整的字串（needle 字符串），顺序和元素都要一一对应。

暴力解法

从主串haystack的pos位置开始一一逐次匹配，每次匹配都要与字串needle依次比较各个字符；发生失配时，从主串的下一个位置与子串的第一个位置开始下一趟匹配。从程序代码上看，需要两个for循环，时间复杂度为O(n*m)。

改进算法

由于存在不必要的回溯，简单字符串匹配算法效率不高：每次匹配，子串都从第一个字符串开始匹配且前一次主串与子串匹配的信息完全丢失。当子串第一个元素匹配成功后，第二个元素匹配失败，第二个元素与第一个元素相同时，第二趟匹配显然是不可能成功的。因此，想要缩减匹配的次数，需要好好利用已经匹配成功的子串的信息，所以引入了几个概念：前缀子串、后缀子串、相等的前缀与后缀子串以及最长相等的前缀与后缀子串。同时，可以用next数组或者prefix table（前缀表）来记录子串信息。

next数组（前缀表）

前缀子串

至少包含主串中第一个字符的子串，例如，”ababa”的前缀子串有”a”, “ab”, “aba”, “abab”

后缀子串

至少包含主串中最后一个字符的子串，例如，”ababa”的前缀子串有”a”, “ba”, “aba”, “baba”

注意：长度为1的字符串没有前缀子串和后缀子串

相等的前缀与后缀子串

存在常数k，使得前k-1前缀子串等于后n-k后缀子串，例如，”ababa”存在两对相等的前缀与后缀子串：”a”, “aba”

最长相等的前缀与后缀子串

在相等前缀与后缀子串中，长度最大的子串，例如，”ababa”的最长相等的前缀与后缀子串为”aba”

什么是next数组或者说前缀表

前缀表，它记录的是子串下标i之前（包括i）的字符串中，最长相等前缀后缀子串长度。它的作用是用来调整子串主串匹配失败时，子串回退的位置信息，子串应该从哪个位置继续开始重新匹配

计算前缀表

对于子串，定义一个for循环，i=1，计算每个子串的最长相同前后缀子串的长度

当前缀表找到主串子串不匹配的时候，子串的指针位置指向前一位的前缀表的值

需要注意的是，主串子串不匹配时，要一直比较子串与主串的值，如果不相等，继续回退；所以这里应该是一个while循环，而不是if判断

while(j > 0 && s[i] != t[j]){
    j = next[j];
}

构造next数组

构造next数组，其实质就是计算子串s，前缀表的过程。一共分为三步：

1、初始化子串位置指针的值

2、处理前后缀不相同的情况

3、处理前后缀相同的情况

1、初始化子串位置指针的值

定义两个指针i，j，j指向前缀末尾位置，i为后缀末尾位置（for循环+1）

int i = 1, j = 0;
next[j] = 0;
//----------------------------------
//遍历子串
for(i = 1; i < s.size(); i++)

2、处理前后缀不相同的情况

当s[i]与s[j]不相同时，也就是前后缀末尾不相同，这个时候需要向前回退，next[j]记录j之前的子串最长相同前后缀的长度

while(j > 0 && s[i] != s[j]){
    j = next[j - 1];
}

3、处理前后缀相同的情况

如果s[i]和s[j]相同，说明找到了相同的前后缀，需要同时向后移动i和j，i随循环移动，j++；最后用next[i]记录相同前后缀的长度

if (s[i] == s[j]) { // 找到相同的前后缀
    j++;
}
next[i] = j;

构建next数组代码

void getNext(int* next, const string& s){
    int j = 0;
    next[0] = j;
    for(int i = 1; i < s.size(); i++) { // 注意i从1开始
        while (j > 0 && s[i] != s[j]) { // 前后缀不相同了
            j = next[j - 1]; // 向前回退
        }
        if (s[i] == s[j]) { // 找到相同的前后缀
            j++;
        }
        next[i] = j; // 将j（前缀的长度）赋给next[i]
    }
}

构造next数组的整个逻辑为：（代码随想录的next数组，初始化第一个元素为-1；按上面的讲法，只需将next数组元素再加1就行）

使用next数组做匹配

在主串s里找是否出现过子串t
定义两个下标，j指向子串起始位置，i指向主串起始位置。
i从0开始，遍历主串，然后比较s[i]和t[j]，如果s[i]与t[j]不相等，j就从next数组里寻找下一个匹配的位置；如果s[i]与t[j]相等，i和j同时向后移动。当j指向子串t的末尾时，说明子串已经完全匹配主串的某个字串。
整体代码如下所示：

int j = 0; 
for (int i = 0; i < s.size(); i++) { 
    while(j > 0 && s[i] != t[j]) { // 不匹配
        j = next[j-1]; // j 寻找之前匹配的位置
    }
    if (s[i] == t[j]) { // 匹配，j和i同时向后移动
        j++; // i的增加在for循环里
    }
    if (j == (t.size() - 1) ) { // 文本串s里出现了模式串t
        return (i - t.size() + 1);
    }
}

整体代码

class Solution {
public:
    void getNext(int* next, const string& s) {
        int j = 0;
        next[0] = 0;
        for(int i = 1; i < s.size(); i++) {
            while (j > 0 && s[i] != s[j]) {
                j = next[j - 1];
            }
            if (s[i] == s[j]) {
                j++;
            }
            next[i] = j;
        }
    }
    int strStr(string haystack, string needle) {
        if (needle.size() == 0) {
            return 0;
        }
        vector<int> next(needle.size());
        getNext(&next[0], needle);
        int j = 0;
        for (int i = 0; i < haystack.size(); i++) {
            while(j > 0 && haystack[i] != needle[j]) {
                j = next[j - 1];
            }
            if (haystack[i] == needle[j]) {
                j++;
            }
            if (j == needle.size() ) {
                return (i - needle.size() + 1);
            }
        }
        return -1;
    }
};
